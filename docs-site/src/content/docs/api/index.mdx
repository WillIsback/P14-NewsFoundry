## Architecture des données

```
Composant React
  └─► Server Action (actions/)
        └─► DAL / Service (service/)
              └─► fetchJson (lib/fetch.lib)
                    └─► Backend FastAPI
                          └─► ServiceResult<T>  ◄── type central (lib/type.lib)
```

Toutes les fonctions DAL retournent `ServiceResult<T>` — une union discriminée sur `ok` qui évite les `try/catch` dans les composants.
