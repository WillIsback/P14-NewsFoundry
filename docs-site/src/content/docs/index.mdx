## À propos

NewsFoundry est une application fullstack qui permet de créer des revues de presse automatisées via un agent RAG (Retrieval-Augmented Generation).

**Frontend :** Next.js 16, React 19, TypeScript, Tailwind CSS 4, shadcn/ui  
**Backend :** FastAPI, LlamaIndex, PostgreSQL  
**Infra :** Railway, Docker, Tailscale
